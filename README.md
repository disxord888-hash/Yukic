# 得点版 & タイマー (Scoreboard)

ブラウザベースのシンプルで高機能な得点版・タイマーアプリです。イベントや配信、競技のタイムキーパーとしてご利用いただけます。

## 機能概要

- **複数プレイヤー対応**: 任意の人数を追加・削除可能なカード形式。
- **スコア機能**: 各プレイヤーごとに「＋」「－」ボタンで得点管理。
- **ハイブリッドタイマー**:
  - **カウントアップ**: 時間無制限の経過時間計測。
  - **カウントダウン**: 秒数を指定しての減産計測（終了時にアラート）。
- **Phase表示**: 画面右下に現在のフェーズ（ラウンド数など）を表示・編集操作可能。
- **カスタマイズ可能な通知**:
  - **カウントダウン時**: 30秒、25秒、20秒、15秒、残り10秒〜0秒は毎秒通知。
  - **定期通知**: 「通知間隔(秒)」で設定した時間ごとに通知音。
- **テーマ切替**: ダークモード / ライトモード対応。
- **サウンド制御**: ワンクリックで消音可能。

## 使用方法

1. **`index.html`** をブラウザ（Chrome, Edge等）で開いてください。
2. **プレイヤーの設定**: 初期状態で2名表示されます。
   - 名前をクリックして編集。
   - 「＋」「－」ボタンでスコアを変更。
   - 下部の「+ プレイヤーを追加」ボタンで人数を増やせます。
   - 右上の「×」ボタンでプレイヤーを削除。
3. **タイマー設定**:
   - **モード選択**: 画面上部の「カウントアップ」「カウントダウン」ボタンで切り替えます。
   - **カウントアップ**: 「通知間隔」を設定すると定期的に音が鳴ります。
   - **カウントダウン**: 「制限時間」を設定します。「通知間隔」も任意で設定可能です。
4. **Phase設定**:
   - 右下の「◀」「▶」ボタンで、数字部分を増減できます。
   - テキストを直接クリックして自由に書き換えることも可能です。
5. **タイマー操作**:
   - 左下のボタン群で操作します（スマホ操作時は左手親指で押しやすい設定）。
   - **STOP**: 計測を一時停止します。
   - **RESET**: タイマーを 00:00 に戻します（確認ダイアログが出ます）。

## キーボードショートカット

マウス操作なしで素早くコントロールできます。

| キー | 動作 |
|---|---|
| **Space** | スタート / ストップ（トグル） |
| **Esc** | リセット |

## 注意事項

- 音声を再生するため、初回使用時は画面上のどこかをクリックする必要がある場合があります（ブラウザの自動再生ポリシーのため）。
- `scoreboard`フォルダ内のファイルを移動・削除しないでください。
